This is the CAPTCHA Breaking Language (CBL) v1.0 syntax highlighting plugin for Notepad++.

Agugust 5, 2012

To use...

   1. Obtain a copy of Notepad++.
   2. Go to "View" > "User Defined Dialogue"
   3. Click "Import" and select the "CBLv1.xml" file.
   4. Go to the "Language" menu and select "CBLv1" at the bottom of the list.
   5. Open a CBL script and enjoy!
